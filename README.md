# MiraiGo-Bot
增强MiraiGo-Bot,开箱可用,降低使用成本

## 功能
* [ ] 使用`sqlite3`存储数据,使用`migrations`同步数据库
* [ ] 实现`miraigo-module`
* [ ] 支持多种类的回复类型
  
  是否实现|类型|编码|描述
  -|-|-|-
  [ ]|文本|text|返回固定文本
  [ ]|随机数|randomMath|返回`[min,max]`中的整数值
  [ ]|随机项|randomItem|返回`[a,b,c,d,e]`中任意个数
  ...|

* [ ] 支持定向回复
* [ ] 支持web端对回复进行配置
* [ ] 支持通过回复进行配置?
* [ ] 缓存支持
